# how-it-works
Steganography works by taking an image and manipulating the LSB (Least Significant Bit) using binary.

# running-the-program
We have a version with and without a GUI overlay. To use the non-GUI version you will need to have an IDE (preferably Visual Studio Code) and to use the GUI version you will need to compile the files into a jar and use something like Launch4j. You can use the non-GUI version and follow the same steps as the GUI version but you will need to click the terminal option in Launch4j so that the program runs through your terminal.

# testing
We have some test images ready for users. Some work while others do not. We do not know and have no plans of figuring out all usable formats, but the program does tell you if it is not compatible.
